About the extension:

  1. GitGoin' only works on the github.com, gitlab.com, bitbucket.org, and dev.azure.com domains.
  2. GitGoin' does not track, store, or transmit any user data. 100% private.
  3. GitGoin' requires certain permissions to work properly, including tabs to create tabs that automatically open the chosen editor's url-scheme, storage for saving preferences, and declarativeContent to detect supported git domains.
  4. You can set the default editor & protocol (git or http) by right-clicking the extension icon and opening the extension options page.
  5. GitGoin' is open-source. For bugs or feature requests, please submit an issue on GitHub or send me a message on Twitter.
